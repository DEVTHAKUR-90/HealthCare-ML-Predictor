# HealthCare ML Predictor 🏥

This project is a Machine Learning-based web application that predicts diseases based on user symptoms.

## 🚀 Features

* Predict diseases from symptoms
* Uses multiple ML models:

  * Naive Bayes
  * Decision Tree
  * Random Forest
* Built using Streamlit

## 🧠 Models Used

* Gaussian Naive Bayes
* Decision Tree Classifier
* Random Forest Classifier

## 📂 Dataset

The main dataset is not included due to its large size.

Download it from here:
👉 https://drive.google.com/file/d/1CrOUwsoHVguL5oHcmtPblt3_6AAJHT9m/view?usp=drive_link

## ⚙️ Installation

git clone https://github.com/DEVTHAKUR-90/HealthCare-ML-Predictor.git
pip install -r requirements.txt

## ▶️ Run

streamlit run m2.py
